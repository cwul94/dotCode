
# extends 활용해 클래스 단일 상속받기

Title : 즐거운 Java 수업

1. 수업을 받는 학생(Student) 클래스, 수업과 관련된 멤버(필드/메소드)
2. 수업을 받는 주체인 나(Chanwool) 클래스, 학생클래스에서 메소드를 상속받아(Overriding) 재정의 + 나만의 특수한 행위 메소드 추가
3. 수업을 받는 주체인 다른학생(Others) 클래스, 또한 학생클래스에서 메소드를 상속받아 재정의 + 다른학생만의 특수한 행위 메소드 추가

목적 : 실제로 동적바인딩이 잘 이루어지고있는지 확인/ 클래스타입 형변환이 잘 이루어지는지 확인 / instanceof 연산자로 타입 확인
